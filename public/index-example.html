🚀 Hướng Dẫn Cài Đặt và Chạy

Cài đặt Node.js

Tải Node.js từ https://nodejs.org

Cài đặt Dependencies 
npm install
Chạy Backend 
npm run server

Server sẽ chạy trên http://localhost:3000

Chạy Frontend (Terminal mới) 
npm run dev

Frontend sẽ chạy trên http://localhost:5173

Hoặc Chạy Cùng Lúc 
npm run dev:full

Cả backend và frontend sẽ chạy cùng một lúc
