❓ COMMON QUESTIONS
════════════════════════════════════════════════════════════════

Q: How do I add more jobs?
A: Edit server.js, find const jobs = [...], add new job objects

Q: How do I change colors?
A: Edit src/styles.css, modify :root variables (lines 1-30)

Q: How do I add new filter?
A: Edit App.jsx and FilterBar.jsx components

Q: How do I deploy?
A: See README.md "Deployment" section

Q: How do I run backend only?
A: npm run server

Q: How do I run frontend only?
A: npm run dev

Q: What if I get errors?
A: Check browser console (F12) and terminal output
