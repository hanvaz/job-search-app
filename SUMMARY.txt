Job Search for Students - Web Application
========================================

📋 PROJECT OVERVIEW
-------------------------------------------
A complete fullstack web application for students to search for job opportunities.
Built with React (frontend) and Node.js/Express (backend).

✅ WHAT WAS CREATED
-------------------------------------------

FRONTEND (React)
- 11 React components for complete UI
- Responsive design (mobile, tablet, desktop)
- Search and filtering functionality
- Favorites management with localStorage
- Job detail modal display
- Loading and error states
- 600+ lines of CSS with modern styling

BACKEND (Node.js/Express)
- Express server with 8+ API endpoints
- 10 sample job listings
- Search, filter, and location-based queries
- CORS enabled for frontend integration
- Complete error handling

UTILITIES & HELPERS
- API call wrapper functions
- localStorage utility functions
- Organized component structure

DOCUMENTATION (6 Files)

CONFIGURATION

- setup.sh and setup.bat for easy setup

🎯 FEATURES IMPLEMENTED
-------------------------------------------

✓ Job Search
  - Real-time search by job title or company
  - Search input with instant filtering

✓ Advanced Filtering
  - Filter by location (Hà Nội, Hồ Chí Minh, Đà Nẵng, Remote)
  - Filter by job type (Thực tập, Bán thời gian, Remote)
  - Combined filtering support

✓ Favorites Management
  - Save/unsave jobs with heart button
  - Persistent storage using localStorage
  - Dedicated favorites tab
  - Quick access to saved jobs

✓ Job Details
  - Modal popup with full job information
  - Description, requirements, and benefits
  - Job metadata (location, type, company)
  - Easy navigation between jobs

✓ User Interface
  - Modern, clean design
  - Responsive layout for all devices
  - Smooth animations and transitions
  - Loading and error states
  - Empty state messages

✓ API Integration
  - 8+ RESTful API endpoints
  - Proxy integration in development
  - Error handling and validation
  - JSON response format

🚀 HOW TO USE
-------------------------------------------

1. INSTALL DEPENDENCIES
   npm install

2. START DEVELOPMENT
   npm run dev:full
   (Runs both backend and frontend)

3. OPEN IN BROWSER
   http://localhost:5173

4. BUILD FOR PRODUCTION
   npm run build
